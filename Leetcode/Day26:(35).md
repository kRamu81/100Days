## **Day 26 of 100 Days LeetCode Challenge**
**LeetCode Challenge #35. Search Insert Position**

```
class Solution {
    public int searchInsert(int[] nums, int target) {
        int left = 0;
        int right = nums.length;
        while(left<right){
            int mid = (left+right)/2;
            if(nums[mid]==target){
                return mid;
            }else if(nums[mid]>target){
                right = mid;
            }else{
                left = mid+1;
            }
        }
        return left;
    }
}
```
